# Job Portal

A full-stack MERN Job Portal application where students can search and apply for jobs, while recruiters can create companies, post jobs, and manage applicants.

## Live Demo

Link: https://job-portal-frontend-jff9.onrender.com  


## Tech Stack

- React.js
- Redux Toolkit
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- Cloudinary
- Render

## Features

- Student and recruiter authentication
- Role-based access
- Job search and filtering
- Company creation and management
- Job posting by recruiters
- Apply to jobs
- Applicant management
- Profile update
- Resume/profile image upload
- Responsive UI

## Deployment

Frontend and backend are deployed on Render.
